# Web 实时监控

## 快速开始

1. 双击 `index.html` 浏览器打开
2. 填入连接信息点击「连接」
3. 底部黑色区域显示连接日志

## 连接信息

| 字段 | 值 |
|------|-----|
| WebSocket 地址 | `wss://g16b99c6.ala.cn-hangzhou.emqxsl.cn:8084/mqtt` |
| 用户名 | EMQX 认证用户名 |
| 密码 | EMQX 认证密码 |
| 群组 ID | `1` |

## 排查

### 连接失败（日志显示 "错误" 或 "断开"）

1. **EMQX 控制台 → 部署详情 → 监听器**
   - 确认有 WebSocket 监听器，端口 8084，状态「运行中」
   - 如果没有：点击「添加监听器」→ 选 WebSocket → 端口 8084 → 启用 TLS

2. **WebSocket 地址格式**
   - 正确：`wss://g16b99c6.ala.cn-hangzhou.emqxsl.cn:8084/mqtt`
   - 注意：`wss://` 不是 `ws://`，`/mqtt` 路径不能少

3. **用户认证**
   - EMQX 控制台 → 访问控制 → 认证 → 确认有用户且密码正确
   - 新建一个 Web 专用用户避免冲突

### 连接成功但无设备

- 确保手机上 App 已运行且 MQTT 已连接
- 确认群组 ID 与 App 中一致（默认 `1`）
